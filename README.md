# Lonomia_genome_assembly
The repository is associated with the manuscript "Genome assembly and venom gene mapping in the medically important moth Lonomia casanarensis (Saturniidae: Hemileucinae)". Authors: Diler Haji1*, Samridhi Chaturvedi2*, Oanh H. Nguyen3, Noravit Chumchim3, Diana M. Toro-Vargas4, Juana Díaz-Díaz4, Lina V. Lozano-Morales4, Gustavo A. Bravo5,6, Camila González4, Noah K. Whiteman2#

Lonomia scripts folder: Contains scripts for analyses of venom genes and contructing the mitochondrial genome.
lonomia_genome_assembly_pipeline_v1.md : Markdown file contains details of the genome assembly using PacBio data. All the slurm scripts used to run this pipeline are provided in this markdown file.
